# Paradise Nursery Shopping Application

## Project Description
Paradise Nursery is a React-based shopping application that allows users to browse and purchase various houseplants. Users can add plants to a cart, update quantities, and view total cost using Redux state management.

## Features
- Browse houseplants by category
- Add products to cart
- Increase/decrease quantity
- View total price dynamically
- Responsive UI for better user experience

## Tech Stack
- React
- Redux Toolkit
- JavaScript
- CSS
